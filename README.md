# PetClicker

A pet-themed clicker game where you nurture a pet by constantly petting it! (only uses left-click as input)

Um, not all features are implemented yet, so it may be a bit boring!! But I'll detail al the features I did implement and what I planned on implementing.

Implemented Features:
- can click pet!
- can use clicks to buy shop items
- shop items include custom mouse cursors (which increase click numbers) and backgrounds
- upload image feature (this unfortunately does not work for the web export but it essentially allows you to upload an image as your pet. In my mind this was cool bc I wanted to upload my friends as pets...hehehe)

To Be Implemented:
- pet stats (happiness, boredom, etc. These would depend on the shop items you buy and how often you click the pet)
- more shop items and functionality
- one key minigames (for stats)
